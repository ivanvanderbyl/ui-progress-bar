Pure CSS Progress Bar
=====================

A Progress Bar implemented entirely using CSS 3 gradients, shadows, and border-radius, with a touch of animation.

Demo: [http://ivan.ly/ui](http://ivan.ly/ui "CSS Progress Bar")
